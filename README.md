# Lumin ClickGUI 开发文档

## 设计理念

**风格**：Apple 极简 + 柔光磨砂 + 无冗余元素  
**关键词**：干净、留白、细线条、弱阴影、微发光、丝滑动画

---

## 配色方案

| 名称 | 颜色值 | 用途 |
|------|--------|------|
| 背景 | `#1A1A1A` | 超纯黑底 |
| 面板磨砂 | `rgba(28, 28, 30, 0.85)` | 面板背景 |
| 分割线 | `rgba(255,255,255,0.1)` | 分隔元素 |
| 主光色 | `#0A84FF` | Apple 原生蓝，强调色 |
| 次光色 | `#5E5CE6` | 淡紫蓝，次要强调 |
| 标题文字 | `#FFFFFF` | 标题 |
| 描述文字 | `rgba(255,255,255,0.6)` | 说明文字 |
| 禁用文字 | `rgba(255,255,255,0.3)` | 禁用状态 |
| 悬浮背景 | `rgba(255,255,255,0.05)` | 悬浮效果 |

---

## 字体系统

| 字重 | 文件 | 用途 |
|------|------|------|
| Light | `PingFangSC-Light.otf` | 说明文字、描述 |
| Regular | `PingFangSC-Regular.otf` | 常规文字 |
| Medium | `PingFangSC-Medium.otf` | 模块名称 |
| Semibold | `PingFangSC-Semibold.otf` | 标题、强调 |

### 📝 步骤 1：配色系统与主题类

**目标**：创建统一的颜色管理类

**文件结构**：
```
dev.lumin.client.gui/
├── theme/
│   └── Theme.java          # 颜色常量定义
```

**Theme.java 内容**：
- 定义所有颜色常量（背景、文字、强调色等）
- 提供 `getColor()` 方法返回 AWT Color
- 提供 `getRGBA()` 方法返回 int 格式颜色

---

### 📝 步骤 2：动画系统

**目标**：实现平滑动画效果

**文件结构**：
```
dev.lumin.client.gui/
├── animation/
│   ├── Animation.java      # 动画基类
│   ├── EaseAnimation.java  # 缓动动画
│   └── AnimationUtil.java  # 动画工具类
```

**动画类型**：
- 淡入淡出（透明度）
- 缩放（打开 GUI）
- 高度展开（设置面板）
- 位置滑动（Toggle、Slider）

**缓动函数**：
- EaseOut（默认）
- EaseInOut
- Linear

---

### 📝 步骤 3：基础组件

**目标**：实现可复用的 UI 组件

**文件结构**：
```
dev.lumin.client.gui/
├── component/
│   ├── Component.java      # 组件基类
│   ├── Toggle.java         # 开关组件
│   ├── Slider.java         # 滑块组件
│   ├── ModeSelector.java   # 模式选择器
│   └── ColorPicker.java    # 颜色选择器
```

**组件设计**：

#### Toggle 开关
- 扁长圆角矩形
- 关闭：灰色描边，空心
- 开启：填充 `#0A84FF`
- 动画：0.2s 平滑滑动

#### Slider 滑块
- 极细轨道
- 进度：主色渐变
- 滑块：小白圆
- 支持拖拽

#### ModeSelector 模式选择器
- 横向小按钮组
- 选中：浅背景 + 主色文字
- 无描边

**文件结构**：
```
dev.lumin.client.gui/
├── element/
│   └── ModuleElement.java  # 模块条目
```

**设计规范**：

- 点击展开：平滑向下展开动画

### 📝 步骤 7：主窗口

**目标**：实现整体布局

**文件结构**：
```
dev.lumin.client.gui/
├── screen/
│   └── ClickGuiScreen.java # 主窗口
```

**设计规范**：
